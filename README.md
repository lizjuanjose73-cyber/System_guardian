# System Guardian

System Guardian es un monitor de sistema desarrollado en Python que analiza el uso de CPU y memoria RAM en tiempo real.

El programa registra métricas del sistema y genera registros con fecha y hora para analizar el comportamiento del sistema.

## Características

- Monitoreo de CPU
- Monitoreo de memoria RAM
- Registro automático de métricas
- Sistema de logs con fecha y hora

## Tecnologías utilizadas

- Python
- psutil
- Git
- GitHub

## Estructura del proyecto

main.py → ejecuta el programa  
monitor.py → obtiene información del sistema  
logger.py → guarda registros del sistema  
config.py → configuraciones del programa  

## Objetivo del proyecto

Este proyecto fue desarrollado como parte de mi formación en programación y aprendizaje de monitoreo de sistemas usando Python.
